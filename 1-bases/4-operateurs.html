JS - Opérateurs

Exercice

Ajouter dans le fichier 4-operateurs.js :

  • Les variables a:10, b:20, c, d, e et réaliser les calculs au fur et à mesure afin d'afficher :
  • (les nombres étant remplacés par les variables et les booléens par des opérations)
  • a : 10 ; b : 20
  • c = a + b => c : 30
  • d = b % 2 => d : 0 ; e = b % 3 => e : 2
  • a += b => a = a + b => a : 30
  • b++ => b : 21
  • a == 30 : true
  • a === 30 : true
  • a == '30' : true
  • a === '30' : false
  • a == c : true
  • a == 30 && b == 21 : true
  • a == 30 && b == 20 : false
  • a == 30 || b == 20 : true
